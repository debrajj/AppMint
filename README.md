# AppMint - Shopify Embedded App

A minimal Shopify embedded app built with React, Polaris, and App Bridge.

## Setup

1. Install dependencies:
```bash
cd web/frontend
npm install
```

2. Run the development server:
```bash
npm run dev
```

## Features

- Shopify Polaris UI components
- App Bridge integration for embedded apps
- Left sidebar navigation with:
  - Dashboard
  - Content
  - Pages
  - Settings
- Dashboard with welcome card and statistics

## Structure

- `App.jsx` - Main app component with navigation and routing
- `main.jsx` - React entry point
- `index.html` - HTML template
- `vite.config.js` - Vite configuration
